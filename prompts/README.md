# Prompt Kits (Pré-prompts)

Biblioteca de pré-prompts para padronizar execução por área do sistema.

## Estrutura

- `prompts/architecture/`
- `prompts/implementation/`
- `prompts/review/`
- `prompts/testing/`
- `prompts/deploy/`
- `prompts/security/`
- `prompts/orchestration/`

## Como usar

1. Escolha o prompt base da área.
2. Preencha os campos `[TODO]`.
3. Execute o fluxo.
4. Salve decisão e evidência no relatório final.
