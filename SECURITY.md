# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 3.x | Yes |
| < 3.0.0 | No |

## Reporting a Vulnerability

Se você identificar uma vulnerabilidade:

1. **Não abra issue pública** com detalhes exploráveis.
2. Envie um relatório privado contendo:
   - descrição do problema;
   - impacto;
   - cenário de reprodução;
   - proposta de mitigação.
3. Aguarde confirmação de recebimento antes de divulgar.

## Scope

Esta política cobre principalmente:

- execução de comandos via tools e wrappers;
- workflows de CI/CD e deploy;
- gerenciamento de segredos e permissões;
- cadeia de suprimentos (dependências e ações).
